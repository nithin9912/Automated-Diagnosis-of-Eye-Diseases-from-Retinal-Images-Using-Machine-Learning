---Automated Eye Disease Diagnosis Using Machine Learning----

Developed a machine learning-based system to detect eye diseases from retinal images.
Classified retinal images as Normal or Glaucoma-Affected.
Implemented image preprocessing and feature extraction techniques for accurate analysis.
Trained and evaluated Decision Tree and Random Forest models for disease prediction.
Built a user-friendly Tkinter GUI for image upload and result visualization.
Automated the diagnosis process, reducing manual effort and improving efficiency.
Enabled faster predictions to support early disease detection and treatment.

Technologies Used: Python, Scikit-learn, Pandas, NumPy, Decision Tree, Random Forest, Tkinter GUI.
